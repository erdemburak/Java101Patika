# Insertion Sort

1. [22,27,16,2,18,6]

- 22<2 : [2 | 27,16,22,18,6]
- 27<6 : [2,6 | 16,22,18,27]
- 16=16: [2,6,16 | 22,18,27]
- 22<18: [2,6,16,18 | 22,27]
- 22=22: [2,6,16,18,22 | 27]
- 27=27: [2,6,16,18,22,27]

2.  Big-O Notation
    O (n^2)

3.  Time Complexity
    Avarage Case : n^2
    Worst Case : n^2
    Best Case : n

4.  18 is in the avarage case because it is in the middle of the array
